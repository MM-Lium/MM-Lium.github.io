新冠肺炎前後對比照曝光！疫情怎麼改變這個世界⋯

2020年3月20日 Yahoo新聞特別企劃

新冠肺炎疫情在全球持續蔓延，各國確診數持續飆高，原本人潮擁擠的觀光景點，如今人煙稀少。路透社拍攝的一系列前後對比圖，帶你來看世界的變化有多大！

佛羅里達州奧蘭多迪士尼主題樂園

2020年整個世界歷經多重磨難，病毒、火災、水災、爆炸都改變了地球原來的風景，Yahoo帶你回顧2020年全球各地曾發生重大事件的新聞現場，對比「前世今生」，每一張照片都是歷史重要的一筆。或許很多事情都已經再也回不去，但也許地球也在蛻變，在不久的將來，可以看到更美的新面貌。
