2020落幕 盤點2021值得關注的全球大事

2020年1月27日 Yahoo新聞特別企劃

新冠肺炎病毒侵襲全球，各國防疫無所不用其極，鎖國封城、居家隔離、經濟斷鏈，還有旅遊、賽事、求學等各種計畫被迫中斷，世界彷彿被按下了暫停鍵。對許多人而言，這一年不僅空白了、消失了，更充斥恐懼與不安，令人亟欲擺脫。而備受期待的2021，除了因疫情而延期的展覽、賽事將再度重啟，政壇、體壇、科技界亦有不少值得關注的大事。

美國新任總統就職 台灣開放美豬進口

2021元旦，台灣開放美豬叩關、放寬美牛進口，不只本土豬農受影響，食安議題也使廣大消費者無法置身事外，從源頭到餐桌諸多配套措施，不得不留心。國際大事則聚焦1月20日，第59屆（45/46任）美國總統拜登宣誓就任，如何有效防堵新冠疫情，各國拭目以待；而台美中的三角關係，也可能隨著白宮易主而出現變化。1月22日，聯合國的《禁止核武器條約》正式生效，儘管已獲50個國家批准，然而全球五大主要擁核國家──美、俄、中、英、法卻未簽署，使條約的實質效力令人存疑。

2021火星年 強國競逐太空

時序推至2月，這個月堪稱「火星月」。美國國家航空暨太空總署（NASA）繼9年前成功發射「好奇號」抵達火星後，再度派遣「毅力號」出馬，預計在2月18日降落，執行火星2020探測車任務。而中國首次火星探測任務「天問一號」，則預定2月11日進入火星軌道、4月著陸，為中國開啟太空探索的全新視野。此外，阿聯酋首次成功發射的「希望號」火星探測器，同樣可望在2月抵達火星。

東京奧運延後一年啟動 各大運動賽事蓄勢待發

2021下半年，許多因新冠疫情而暫緩的國際大事將重新運作，同樣令人引頸企盼。首先是體壇盛事──2020東京奧運，7月23日將揭開序幕。東奧團隊表示，若屆時仍壟罩在疫情陰霾之下，或許開、閉幕式將會簡化，各國工作人員、代表團與觀賽民眾也可能縮減，但將如期舉辦、不再延期。為了配合東奧延期，原定8月登場的成都世大運也順延兩日，於18日開賽，讓選手們盡可能兼顧兩邊賽事。

香港衝突未歇 立法會選舉測民心

此外受疫情影響的，還有香港第七屆立法會選舉。原定於2020年9月舉行，由地方選區和功能界別各選出35名議員，遭到香港特區行政長官林鄭月娥以防疫為由而推遲一年；這也是香港回歸中國23年來，首次延後立法會選舉日期，掀起不小輿論聲浪。而粉絲盼了三年的杜拜世界博覽會，同樣因疫情考量確定再延一年，改至2021年10月1日至2022年3月31日期間舉行。

歷史傷痕仍須撫平 中華民國110歲生日快樂

2021向前看的同時，別忘了還有不少歷史大事值得回顧。3月11日，奪走15,881 人的日本三一一大地震，將屆滿10週年，這場浩劫至今仍令許多日本人餘悸猶存；而被美國視為「永遠的噩夢」的九一一恐怖襲擊事件，一晃眼也來到20週年，雙子星大樓倒塌畫面卻依舊歷歷在目。雖有悲、亦有喜，10月10日，台灣將迎來中華民國誕生第110週年，雙十國慶大典格外令人期待。而中國共產黨也在7月23日正式成立100週年，官方為了盛大慶祝，已提前一年向各界邀稿、徵畫、開拍電影，屆時將全方位宣傳創黨百年。
Yahoo奇摩推出「2020年度回顧專題」，從武漢封城、新冠肺炎造成的悲苦離散，到各國大選的政治更迭、Kobe墜機等名人相繼離去...如果2020是一本書，你是否又哭又笑地讀完？觀後感想又是什麼？無論答案如何，Yahoo奇摩陪你翻過這一頁，一起迎接新希望，就讓各種值得期待的小確幸，填滿2021新篇章。

2021後疫情時代 投資該如何安全布局？

2020年11月26日 週四

為何漲不停？

2020年台股歷經了疫情肆虐，行情卻在一片悲觀中上演絕地大反攻，由破底轉為頻頻創新高。投資人還沒來的及理清頭緒，股票市場一轉眼已經漲到歷史新高位了。
大多數投資人心裡不禁納悶，
「疫情受控了嗎？」
「景氣回溫了嗎？」
他們想問，到底在漲什麼？
事後回顧，疫情至今仍未受控，流失的人潮也未回流，失去的工作機會、被迫改變的產業生態，恐怕也都回不去了。正是由於經濟面臨前所未有的嚴峻環境，於是各國政府啟動了一系列的人為干預，大量印鈔成為了挽救經濟緊急手段。

廣泛印鈔的影響

人民沒錢，所以政府發錢，看似皆大歡喜，沒有輸家。不過這些大量白花花憑空蹦出來的鈔票，正在改變這個世界運行的規則。當所有人都不靠勞務得到一筆錢，那不是人民變有錢了，而是每一塊錢，能買到的東西變少了，包含看得到的實體商品，和看不到的虛擬商品，例如股票市場。
股票市場裡，大家都知道買股要買績優股，所以績優股原本就是眾人追捧的標的。現在好了，大量的鈔票湧入市場，買的不會是殭屍股，也不會是雞蛋水餃股，還是更加大力度地買進績優股。
所以今年以來，投資人可以發現，原本就買不起的績優股，漲得更快、變得更貴。而原先的低價股，除了少數投機股以外，哪怕是當前股市創新高了，多數至今仍然沒有太大改變。
關鍵就在於這些大量湧入市場的錢是「聰明錢」，它們鎖定的會是「績優股」、「成長股」，恰好台股占比最高的龍頭股台積電就是全球最知名的「成長股」，這也給台股創新高帶來了最合理的解釋。

政府干預不會停止

大家都知道，疫情改變了「舊生態」，帶來了「新常態」。資本市場以漲跌表現給了明確方向，踏上風口的產業一飛沖天，受疫情衝擊的產業相形黯淡。投資人適應得很快，因為它們永遠聚焦在市場焦點。
不過就業市場要由「舊生態」轉為「新常態」，並不像是投資人「換股操作」這麼無痛簡單。
「新常態」取代「舊生態」，並不代表因為疫情而失業的勞工能夠找到新工作，社會資源該如何配置？中間的「陣痛期」會多久？沒人能肯定。一但經濟「喊痛」，那就需要「止痛」，於是大規模的印鈔救市成為了可預期的選擇。
在可預見的未來，政府干預不會停止，湧入市場的「聰明錢」，也會不斷地找尋投資機會。所以績優成長股，將享有更高的「本益比」、甚至是從未達到的「本夢比」境界。

投資人該如何布局？

既然台股最具代表性的「成長股」就是台積電，在投資組合內納入台積電，幾乎是台股股民的必備基本持股。
問題是目前台積電動輒4、5百塊，一般投資人恐怕也買不下手，最簡單的方法，就是透過「零股方式」或「買進ETF間接持有台積電」作為主要手段。
尤其是買進ETF來間接持有，如此一來不僅能買進台積電，更能透過ETF的內建機制來自動挑選一籃子「成長股」。

不想費神挑選個股的投資人， ETF將是事半功倍的選擇。

現在進場不怕追高嗎？

展望2021，紙幣超發的時代，投資人當然可以選擇保守地把所有的錢存在銀行裡，等待股市修正後再投入。問題在於修正多少算修正？而修正開始到結束的時間又要經歷多久？這當中包含了人為的判斷與不確定性。
最重要的是，手中的鈔票也正在這個過程中快速貶值，這也正是大舉印鈔要承擔的「後遺症」。
為避免持有的現金縮水，轉而擁抱「保值資產」成了少數出路，這也是投資人應該持續抱股的理由。
如果害怕買高、買貴了，不妨採用定期定額的方式投入ETF，除了能避免主觀因素影響投資績效，也能以此抵銷逐漸流失的鈔票價值。
